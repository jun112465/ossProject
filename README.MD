# ossProject
대학교에서 오픈소스sw라는 과목을 수강했는데 
과제가 오픈소스를 활용해 자유 주제로 앱을 만드는 것이였다다. 
그래서 이렇게 서로 다른 사람의 사람들이 일정을 공유하는 앱을 만들어봤다.

React-native를 이용해 앱을 만들고
Spring-boot를 이용해 서버를 만들었다.

이 레포지토리는 react-native 코드를 포함한다.
